> ## **设备点检维修 Demo APP**

---

**设备点检维修是掌握设备资产使用状态状况，线上记录设备维修过程，形成企业设备维修档案，有助于企业规范维修作业行为、主动分析发现设备故障，提高设备利用率。**

**我们将针对设备点检工单的CURD(增删改查)、维修记录的CURD(增删改查)、工单受理、工单关闭、消息推送等页面功能，进行案例说明。**
