> ## **设备台账 Demo APP**

---

**设备台账是掌握企业设备资产状况，反映企业各种类型设备的拥有量、设备分布及其变动情况的主要依据。**

**我们将针对设备资产信息的CURD(增删改查)，进行案例说明，如何结合对象脚本以及页面组态加低代码开发APP。**
